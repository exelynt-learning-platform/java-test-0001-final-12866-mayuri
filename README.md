# Java Pattern Program

## Overview

This project contains a Java program that prints a symmetric star pattern using nested loops.
The goal of the program is to demonstrate understanding of loop structures, spacing control, and pattern construction in Java.

## Problem Statement

Write a Java program that prints the following pattern:

```
    *
   * *
  *   *
 *     *
*       *
 *     *
  *   *
   * *
    *
```

## Concepts Used

* Java nested loops
* Conditional statements
* Spacing and alignment control
* Console output formatting

## Program Logic

1. The pattern is divided into two parts:

    * Upper half (increasing internal spaces)
    * Lower half (decreasing internal spaces)
2. Nested loops are used to:

    * Print leading spaces
    * Print stars (`*`)
    * Control inner spacing between stars
3. The program ensures the pattern remains symmetrical.

## How to Run

1. Compile the program:

```
javac PatternProgram.java
```

2. Run the program:

```
java PatternProgram
```

## Expected Output

```
    *
   * *
  *   *
 *     *
*       *
 *     *
  *   *
   * *
    *
```
